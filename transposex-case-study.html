Transpose X: Music Transcription Case Study

Proof-of-Concept Mobile App with OMR Integration

Problem

Musicians often need to digitize sheet music, but manual transcription is tedious and error-prone.

Solution

Built a prototype iOS app with a Node.js backend that integrates Audiveris OMR to convert sheet music images into editable MusicXML files.

Technologies

  • Swift (iOS app)
  • Node.js + Express (backend)
  • Audiveris (Java OMR engine)
  • Docker for backend orchestration

Challenges

  • Java version compatibility with Audiveris.
  • Native library integration in a Dockerized cloud environment.
  • Handling file uploads and efficient processing pipelines.

Results

  • Demonstrated full workflow: capture sheet music ‚Üí backend OMR ‚Üí export MusicXML.
  • Scalable Dockerized backend deployable to cloud services.
  • Functional prototype UI with upload, progress tracking, and export features.
